Sebagai mahasiswa semester 2 jurusan sistem informasi, tentu saya memiliki pengalaman programming yang bisa diceritakan. Saya memulai coding ketika saya diterima di kuliah, pada saat itu saya awalnya belajar python, belum tuntas, dosen saya pada pertemuan kuliah pertama menyuruh mahasiswanya untuk menggunakan Java sebagai bahasa pemrograman semester pertama, hal itu membuat saya beralih dari Python ke Java. Sekitar 1 semester menggunakan Java dasar hingga OOP, saya beralih menggunakan PHP karena pada saat itu saya rencana mau mengikuti bootcamp di Sanbercode kelas laravel. Saya tidak merasa kesulitan ketika belajar PHP dibandingkan Java, mungkin itu karena paradigma pemrograman saya sudah terbentuk sehingga pindah dari satu bahasa ke bahasa lain menjadi mudah. Saya mempelajari PHP dasar hingga OOP, setelah itu belajar menggunakan framework laravel di kelas Sanbercode. 2 bulan saya tuntaskan belajar laravel dari level dasar hingga advance di Sanbercode. Pada bulan kedua belajar Laravel saya juga belajar Javascript karena pada saat itu saya mengambil kelas Laravel VueJS. Setelah belajar Laravel saya melanjutkan 1 bulan untuk mempelajari Javascript tingkat lanjut melalui kelas di Sanbercode yaitu NodeJS Backend Adonis, disitu saya juga mempelajari Typescript, dimana tingkat kesulitannya lebih sulit dibandingkan Javascript. Jadi pada kesimpulannya bahasa pemrograman yang telah saya kuasai ada empat yaitu Java, PHP, Javascript, dan Typescript. Untuk Framework yaitu Laravel, AdonisJS, NodeJS, dan sedikit VueJS. Maka dari itu saya ingin meningkatkan keahlian VueJS saya dengan mengikuti kelas ini.
